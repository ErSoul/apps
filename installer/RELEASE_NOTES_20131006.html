KaOS - 2013.10

While KaOS is under development, expect bi-weekly ISOs to test updates, changes, different features. Understand that due to these changes some regressions might be introduced, making the installation of this distrubution not the best option at the moment when a complete stable system is needed. Once installed though, KaOS is a very reliable distribution.

KDE is at 4.11.2

GFXboot is included with KaOS artwork, Grub theme monochrome.

Repositories, linux 3.10.11, linux-next 3.11.6, systemd 108, kmod 15, networkmanager 0.9.8.8, calligra 2.7.4, libreoffice 4.1.2, chrome 32.0, firefox 24.0, clementine 1.2.0, ardour 3.5
Complete language packs are now available for KDE, Calligra, Firefox, Libreoffice and Thunderbird.

GUI for package management is Octopi.

To create reliable installation media, please follow the instructions the downlaod page. KaOS's ISO's do not support unetbootin, and DVDs need a burn speed no higher than 4x.
