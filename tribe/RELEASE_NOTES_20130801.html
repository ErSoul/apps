KDE is at 4.11RC2.

GFXboot is included with KdeOS artwork, Grub theme momochrome.

Repositories, linux 3.9.8, linux-next 3.10.4, systemd 106, kmod 14, networkmanager 0.9.8.2, calligra 2.7.1, libreoffice to 4.1.0, chrome 30.0.1573.2, firefox 22.0.
The much anticipated Steam has been added to the games repositories.

GUI for package management is Octopi.

To create reliable installation media, please follow the instructions the downlaod page. KdeOS's ISO's do not support unetbootin, and DVDs need a burn speed no higher than 4x.
